# lucablanchi.it

Static personal draft site built with Astro.

## Requirements

- Node.js 22.12.0 or newer
- npm

If you use `nvm`, run:

```sh
nvm use
```

## Scripts

```sh
npm install
npm run dev
npm run build
npm run preview
```

The build outputs static files to `dist/`, preserving the existing `.html`
draft URLs.
